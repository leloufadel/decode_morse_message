# MORSE MESSAGE <a name="about-project"></a>

The "Decode Morse Code" program is a straightforward tool for deciphering Morse code. It decodes individual Morse code characters, converts them to uppercase letters, translates entire Morse code words with spaces between characters, and deciphers entire Morse code messages with three spaces between words.

## 💻 Getting Started <a name="getting-started"></a>


To get a local copy up and running, follow these steps.

### Prerequisites

In order to run this project you need:

Morse-Mesaage installed in your machine.

### Setup

Clone this repository to your desired folder:


```sh
  git clone https://github.com/leloufadel/decode_morse_message.git
  cd decode_morse_message
```


### Usage

To run the project, execute the following command:

```sh
  rails server
```
